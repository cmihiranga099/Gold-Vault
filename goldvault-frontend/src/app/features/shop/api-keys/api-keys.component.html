Quick API reference

Base URL: http://localhost:8080/api/v1/pos

GET /customers List all customers
GET /customers/nic/{nic} Look up customer by NIC
POST /customers Register a new customer
GET /tickets?status=ACTIVE List tickets (filter by status)
GET /tickets/number/{ticketNum} Get ticket by ticket number
POST /tickets Grant a new pawn ticket
PUT /tickets/{id}/redeem Redeem a ticket
POST /payments Record a payment
GET /tickets/{id}/payments Get payment history

Example request (curl):

curl -X GET http://localhost:8080/api/v1/pos/tickets \
  -H "X-API-Key: gv_YOUR_API_KEY_HERE" \
  -H "Content-Type: application/json"
